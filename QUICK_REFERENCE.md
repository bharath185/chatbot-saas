## ⚡ Get Started in 3 Steps

### Step 1: Setup (30 min)
```bash
# Clone project
git clone <your-repo>
cd chatbot-saas

# Create .env with Claude API key
echo "ANTHROPIC_API_KEY=sk-ant-your-key" > .env

# Run locally
bash quickstart.sh
```

Open:
- Dashboard: http://localhost:3000
- API: http://localhost:5000

### Step 2: Deploy (30 min)
```
Backend: Deploy app.py to Render.com (free)
Frontend: Deploy ChatbotSaaS.jsx to Vercel.com (free)
Payments: Add Razorpay code from PAYMENT_INTEGRATION.md
```
