## ⚡ Quick Start (5 Minutes)

### Local Development

```bash
# 1. Clone or download this project

# 2. Get Claude API key (FREE CREDITS included!)
# Go to https://console.anthropic.com/account/keys
# Create new API key and copy

# 3. Create .env file with your key
ANTHROPIC_API_KEY=sk-ant-your-key-here

# 4. Run one command
bash quickstart.sh

# That's it! Backend runs on :5000, Frontend on :3000
```

### First Deployment (30 Minutes)

```bash
# Backend → Render (free) in 5 minutes
# 1. Push code to GitHub
# 2. Go to render.com → New Web Service
# 3. Connect repo, add .env variables
# 4. Deploy!

# Frontend → Vercel (free) in 5 minutes
# 1. Go to vercel.com → Import project
# 2. Connect GitHub repo
# 3. Set REACT_APP_API_URL to your Render backend
# 4. Deploy!

# Payment → Razorpay (₹0 setup)
# 1. Sign up at razorpay.com
# 2. Add code from BUSINESS_GUIDE.md
# 3. Start accepting payments!
```
